# DevPulse Backend

AI-Powered Engineering Workflow Automation Backend.

## Prerequisites

- Go 1.21+
- PostgreSQL
- Make

## Getting Started

1.  **Install dependencies:**
    ```bash
    make tidy
    ```

2.  **Run the server:**
    ```bash
    make start
    ```

3.  **Build for Mac:**
    ```bash
    make build-mac
    ```
